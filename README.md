## 📸 Sobre el proyecto

**GoalKeeperNow** es una plataforma web diseñada para conectar equipos de fútbol con porteros disponibles de forma rápida y sencilla ⚽🥅.

La aplicación permite que los porteros creen perfiles con información como ubicación, nivel de experiencia, disponibilidad y precio por partido. Por otro lado, los equipos pueden buscar arqueros cercanos, visualizar perfiles y enviar solicitudes para completar sus partidos de fútbol amateur.

El proyecto busca solucionar un problema común en el fútbol recreativo: la falta de porteros disponibles al momento de organizar un partido.
